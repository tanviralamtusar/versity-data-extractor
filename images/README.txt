   ======================================
   1. Install sharp (npm install sharp)

   3. It will create all the required PNG files

   RECOMMENDED ICON DESIGN
   =======================
   - Use colors: Purple (#667eea) or similar
   - Simple design with API/network symbols
   - Icon should be recognizable at 16x16 pixels
